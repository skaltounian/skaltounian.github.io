Establishing The Colour Scheme for This Website

Introduction

This article describes how I developed the colour scheme for this website.

Discussion

I used the Adobe Color tool to develop the colours for this page. It was pretty easy to do.

  • Navigated to the Adobe Color website
  • From the home page I navigated to the Create option
  • Used the Extract Theme option on the website
  • I dragged and dropped an image from my collection into the work area
  • The tool gerated a set of colours for me to use for my website

Once I had the colours, I had to do some experimentation to choose the combinations for the various elements.
